# UBLESTRAMK v0.9.0-beta Release Notes

## Installation
1. Download `UBLESTRAMK-v0.9.0-beta.zip`
2. Install via Magisk/KernelSU/APatch Manager
3. Reboot your device
4. Configure denylist/unmount modules for target apps
5. Clear target app data

## What's Included
- Full module source code
- Flashable ZIP (12KB)
- Zygisk native module source (C++)
- Build system
- Documentation

## Beta Notice
This is a beta release. Please report issues at:
https://github.com/lee-muriithi-kingori/UBLESTRAMK/issues

Include:
- Device model
- Android version
- Root solution and version
- Target app name
- Log file contents

## Target Apps
- Uber (Driver, Rider, Eats)
- Banking apps (Chase, BofA, Wells Fargo, Citi, PayPal, Venmo)
- Crypto (Coinbase, Binance)
- Streaming (Netflix, Disney+, Hulu)
- Games (Pokemon GO)

## Important Setup
- **Magisk**: Enable Zygisk, turn OFF "Enforce DenyList", add apps to DenyList
- **KernelSU**: Install ZygiskNext, enable "Unmount modules" for target apps
- **APatch**: Install ZygiskNext, enable "Unmount modules" for target apps
